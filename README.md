## 🗺️ Mapa de Contenidos

Esta guía está organizada de manera modular para que puedas estudiar a tu propio ritmo. Cada módulo cuenta con explicaciones detalladas, diagramas de flujo, **ejercicios prácticos con soluciones** y **errores comunes analizados**.

### 📚 Módulos Teórico-Prácticos
- [Módulo 0 - Mentalidad AI-First](modulos/00-mentalidad-ai-first/README.md) 🧠 (El cambio de paradigma y flujos de trabajo).
- [Módulo 1 - Conceptos Fundamentales](modulos/01-conceptos-fundamentales/README.md) 📖 (LLMs, tokens, comparativa de modelos y casos por industria).
- [Módulo 2 - Herramientas AI para Desarrolladores](modulos/02-herramientas/README.md) 🛠️ (Cursor, Copilot, CLI de Claude y stacks recomendados).
- [Módulo 3 - Prompt Engineering](modulos/03-prompt-engineering/README.md) ✍️ (Zero-Shot, Few-Shot, CoT y prompts listos para copiar).
- [Módulo 4 - Integración con APIs de AI](modulos/04-apis/README.md) 🔌 (Conexión de APIs, streaming, rate limits y presupuestos de costos).
- [Módulo 5 - Proyectos Prácticos y Vibe Coding](modulos/05-proyectos/README.md) 🚀 (Vibe coding paso a paso con Lovable y v0).
- [Módulo 6 - AI Agents y Automatización](modulos/06-agentes/README.md) 🤖 (Agentes autónomos, CrewAI y LangChain).
- [Módulo 7 - Construir Productos con AI](modulos/07-productos/README.md) 📦 (Arquitectura de producción, seguridad y monitoreo).
- [Módulo 8 - RAG y Memoria en AI](modulos/08-rag/README.md) 🗄️ (Embeddings, bases de datos vectoriales y ChromaDB).
- [Módulo 9 - Fine-tuning y Modelos Propios](modulos/09-fine-tuning/README.md) 🎛️ (Datasets en JSONL y entrenamiento fino vs RAG).
- [Módulo 10 - Ética, Seguridad y Buenas Prácticas](modulos/10-etica/README.md) 🛡️ (Prompt injection, protección de datos PII y rate limiters).

## 🏗️ Proyectos Reales Ejecutables

Encuentra código fuente de Python funcional, documentado y listo para ser probado en local en la carpeta [proyectos/](proyectos/):

1. **[01-cli-traductor](proyectos/01-cli-traductor/)**: Traductor de código multi-lenguaje de consola usando Claude.
2. **[02-generador-readme](proyectos/02-generador-readme/)**: Analizador estático de código local que genera automáticamente la documentación de tu repo.
3. **[03-chatbot-memoria](proyectos/03-chatbot-memoria/)**: Chatbot interactivo de terminal con persistencia de historial de chat en JSON.
4. **[04-api-fastapi](proyectos/04-api-fastapi/)**: API REST construida con FastAPI que expone servicios de auditoría y generación de código.
5. **[05-sistema-rag](proyectos/05-sistema-rag/)**: Pipeline RAG local usando ChromaDB para indexar archivos `.txt` y responder preguntas grounded.
